How It Works

1

Install

Download and install the Cart Cookie Deleter extension from this page or the Chrome Web Store.

2

Click

Navigate to any Shopify store and click the extension icon in your browser toolbar.

3

Clear

The "cart" cookie is instantly deleted and the page refreshes automatically.

Installation Instructions

  1. Download & Extract

    After clicking the download button, save the clearcart.zip file to your computer and extract its contents to a folder.

  2. Open Chrome Extensions

    Open Chrome and type chrome://extensions/ in the address bar, or go to Menu → More Tools → Extensions.

  3. Enable Developer Mode

    Toggle on "Developer mode" in the top-right corner of the Extensions page.

  4. Load Unpacked

    Click the "Load unpacked" button that appears, then navigate to and select the folder where you extracted the extension files.

  5. Pin the Extension

    Click the puzzle piece icon in Chrome's toolbar, find "Shopify Clear Cart", and click the pin icon to keep it visible in your toolbar.
